# Intake Workflow

## New Lead

Required fields:

- Name
- Contact method
- Service needed
- Location or service area, if relevant
- Timeline

## Appointment Request

Required fields:

- Name
- Contact method
- Requested date or time window
- Reason for appointment

## Existing Customer

Required fields:

- Name
- Contact method
- Existing issue or account reference, if safely available
- Summary of request

## Human Callback

Required fields:

- Name
- Best callback method
- Short reason for callback
- Urgency level
