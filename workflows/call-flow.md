# Call Flow

## 1. Greeting

The receptionist greets the caller, identifies the business, and asks how it can help.

## 2. Intent Detection

The caller is routed into one of these paths:

- New lead
- Existing customer
- Appointment request
- Pricing question
- Support issue
- Human callback request

## 3. Intake

The receptionist collects the minimum required details for the selected path.

## 4. Confirmation

The receptionist repeats the important information and asks the caller to confirm it.

## 5. Handoff

The receptionist prepares a webhook payload for follow-up by a backend, CRM, calendar, or human review queue.
